About LeAFtool
==============


Macro for Explorer tool develop by IPSDK
https://www.reactivip.com/fr/traitement-dimages/#graphic

Install
=======

.. code-block:: bash

    cd .local/ReactivIP/Explorer/Macro_Interface/
    git clone https://github.com/sravel/LeAFtool.git
